Attaboy!
========

Get stuff done and feel good doing it.

Dependencies
============

    npm install express mysql@2.0.0-alpha8

This project also requires a config.js file, it should look like this:

    module.exports = {
        port: 8080, // Node's listening port
        database: {
            user: '',
            password: '',
            database: ''
        }
    };
